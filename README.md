# Bank Loan Approval Prediction

This project predicts whether a bank loan application will be approved based on applicant details such as income, education, marital status, credit history, loan amount, and property area.

## Project Overview

The goal of this project is to build a machine learning classification model that can predict loan approval status using historical applicant data.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Workflow

1. Loaded the loan prediction dataset
2. Checked dataset information and missing values
3. Handled missing values using mean and mode
4. Encoded categorical features using Label Encoding
5. Scaled numerical features
6. Split the dataset into training and testing sets
7. Trained a Random Forest Classifier
8. Evaluated the model using accuracy, confusion matrix, and classification report
9. Predicted loan approval for sample data

## Model Used

Random Forest Classifier

## Evaluation Metrics

- Accuracy Score
- Confusion Matrix
- Precision
- Recall
- F1-Score

## Output

The model predicts whether a loan application is:

- Approved
- Rejected

## Conclusion

This project demonstrates a complete beginner-friendly machine learning workflow for solving a real-world classification problem.
